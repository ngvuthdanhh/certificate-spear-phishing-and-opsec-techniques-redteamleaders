# Operator Checklists

A set of high-level safety and quality checklists for educational purposes.

---

## 1. Pretext Quality Checklist
- Matches real workflow  
- Tone is consistent  
- Minimal necessary detail  
- Contains clear context  
- Avoids overcomplicated narrative  

---

## 2. OPSEC Risk Checklist
- No unique linguistic quirks  
- No metadata leaks  
- Consistent infrastructure separation  
- Timezone obfuscation maintained  
- Cover story aligned  

---

## 3. Recon Source Checklist
Only permissible sources:

- Public corporate data  
- Job postings  
- Press releases  
- Vendor documentation  
- Public-facing systems  

(No personal data scraping.)
