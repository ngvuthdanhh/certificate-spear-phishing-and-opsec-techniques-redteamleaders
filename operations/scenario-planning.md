# Scenario Planning
This file provides structured planning templates for defensive spear-phishing scenarios used in awareness training.

---

## Objectives
- Simulate realistic phishing workflows  
- Identify detection points  
- Strengthen defensive readiness  

---

## Scenario Structure
1. **Context**  
   What workflow or department is being simulated.

2. **Trigger**  
   Event prompting the message (update, invoice, policy change).

3. **Message Elements**  
   - Tone  
   - Signature format  
   - Pretext justification  

4. **Red Flags for Defenders**  
   - Language anomalies  
   - Timing mismatch  
   - Inconsistent sender roles  

---

## Deliverables
- Scenario description  
- Expected defender response  
- Awareness checklist
